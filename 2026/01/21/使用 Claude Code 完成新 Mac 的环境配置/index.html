使用 Claude Code 完成新 Mac 的环境配置

Jaryujun
2026-01-21
Claude Code, Homebrew, Mac

本文提供了最简化的 Mac 开发环境配置方案：通过一个 AI 提示词，让 Claude Code 自动完成从 Homebrew 到应用安装再到系统配置的所有工作。

准备工作

安装 Claude Code CLI

在终端中执行以下命令：

brew install --cask claude-code

如果还没有安装 Homebrew，先安装它：

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
brew install --cask claude-code

启动 Claude Code：

claude

配置提示词

在 Claude Code 中复制并执行以下完整提示词：

🤖 Mac 环境一键配置提示词

你好！请帮我完成新 Mac 的完整开发环境配置。每完成一步都要进行验证，确保安装成功后再继续下一步。

## 第一部分：安装和配置 Homebrew

1. 检查 Homebrew 是否已安装
   - 执行：brew --version
   - 如果未安装，执行安装命令：
     /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   - 配置环境变量到 ~/.zprofile：
     echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
     eval "$(/opt/homebrew/bin/brew shellenv)"

2. 验证 Homebrew 安装
   - 执行：brew --version
   - 显示版本号

3. 更新 Homebrew
   - 执行：brew update

---

## 第二部分：安装和配置开发环境

### 2.1 安装 Git

1. 安装 Git
   - 执行：brew install git

2. 验证 Git 安装
   - 执行：git --version
   - 显示版本号

3. 提醒我配置 Git 用户信息（稍后手动执行）：
   git config --global user.name "Your Name"
   git config --global user.email "your.email@example.com"

### 2.2 安装 JDK 17

1. 安装 OpenJDK 17
   - 执行：brew install openjdk@17

2. 创建符号链接
   - 执行：sudo ln -sfn /opt/homebrew/opt/openjdk@17/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-17.jdk

3. 验证 JDK 安装
   - 执行：/opt/homebrew/opt/openjdk@17/bin/java -version
   - 显示 Java 版本号

### 2.3 安装 Node.js 和 pnpm

1. 安装 Node.js
   - 执行：brew install node

2. 验证 Node.js 安装
   - 执行：node --version
   - 执行：npm --version
   - 显示版本号

3. 安装 pnpm
   - 执行：npm install -g pnpm

4. 验证 pnpm 安装
   - 执行：pnpm --version
   - 显示版本号

### 2.4 安装 Maven

1. 安装 Maven
   - 执行：brew install maven

2. 验证 Maven 安装
   - 执行：mvn --version
   - 显示版本号

### 2.5 配置环境变量

1. 检查 ~/.zshrc 是否已包含环境变量配置
   - 如果没有，添加以下内容到 ~/.zshrc：

   # Java 环境变量
   export JAVA_HOME=/opt/homebrew/opt/openjdk@17
   export PATH="$JAVA_HOME/bin:$PATH"

   # Maven 环境变量
   export MAVEN_HOME=/opt/homebrew/opt/maven
   export PATH="$MAVEN_HOME/bin:$PATH"

   # Node.js 环境变量
   export NODE_HOME=/opt/homebrew/opt/node
   export PATH="$NODE_HOME/bin:$PATH"

   # npm 全局包路径
   export PATH="$HOME/.npm-global/bin:$PATH"

   # pnpm 环境变量
   export PNPM_HOME="$HOME/Library/pnpm"
   export PATH="$PNPM_HOME:$PATH"

2. 重新加载配置
   - 执行：source ~/.zshrc

3. 验证环境变量
   - 执行：echo $JAVA_HOME
   - 执行：echo $MAVEN_HOME
   - 执行：echo $NODE_HOME
   - 执行：echo $PNPM_HOME
   - 执行：java -version
   - 执行：mvn -version
   - 执行：node --version
   - 执行：npm --version
   - 执行：pnpm --version
   - 确认所有命令都能正常工作

## 第五部分：修改系统偏好设置

### 5.1 配置 Dock

1. 设置 Dock 自动隐藏延迟为 0
   - 执行：defaults write com.apple.dock autohide-delay -float 0

2. 删除动画时间修改器（保持默认动画）
   - 执行：defaults delete com.apple.dock autohide-time-modifier

3. 重启 Dock
   - 执行：killall Dock

4. 验证 Dock 设置
   - 执行：defaults read com.apple.dock autohide-delay
   - 应该显示：0

### 5.2 配置 Fn 键

1. 设置 Fn 键为标准功能键模式
   - 执行：defaults write NSGlobalDomain com.apple.keyboard.fnState -bool true

2. 验证 Fn 键设置
   - 执行：defaults read NSGlobalDomain com.apple.keyboard.fnState
   - 应该显示：1

### 5.3 配置触控板速度

1. 设置触控板移动速度为 2.0
   - 执行：defaults write NSGlobalDomain com.apple.trackpad.scaling -float 2.0

2. 验证触控板速度
   - 执行：defaults read NSGlobalDomain com.apple.trackpad.scaling
   - 应该显示：2

### 5.4 配置触控板点按力度

1. 设置触控板点按力度为弱点按
   - 执行：defaults write com.apple.AppleMultitouchTrackpad FirstClickThreshold -int 0
   - 执行：defaults write com.apple.AppleMultitouchTrackpad SecondClickThreshold -int 0

2. 验证触控板点按力度
   - 执行：defaults read com.apple.AppleMultitouchTrackpad FirstClickThreshold
   - 应该显示：0

配置完成后

手动完成的任务

  1. 配置 Git 用户信息

    git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

  2. 启动 Docker（可选）

    • 打开 Docker Desktop 应用
    • 等待 Docker 引擎启动完成
    • 验证：docker --version

  3. 初始化 Podman（可选）

    podman machine init
podman machine start
podman machine list

  4. 重启系统

    • 重新登录或重启 Mac，使 Fn 键和触控板设置生效
