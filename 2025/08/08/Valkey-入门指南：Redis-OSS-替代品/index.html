一、Valkey 简介

Valkey 是一款开源的 Redis 替代品，它提供了与 Redis 相似的 API 和功能，但具有更高的性能和更低的成本。Valkey 的架构设计非常独特，它将 Redis 的内存存储和持久化功能分离，使得 Valkey 可以同时支持内存存储和持久化存储。
